## 写一个Yaml

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: nginx-pod
spec:
    containers:
    - name: nginx
      image: nginx:1.8
```



## 部署Pod

```shell
kubectl apply -f nginx_pod.yaml

pod/nginx-pod created
```

提示创建成功



## 查询Pod

```shell
kubectl get pods

NAME                               READY   STATUS              RESTARTS   AGE
nginx-pod                          1/1     Running             0          4m22s
```

打印出状态：

- NAME - nginx-pod就是对应上面 `metadata.name`
- READY - 就绪的个数
- STATUS - 当前的状态，RUNNING表示运行中
- RESTARTS - 重启的次数
- AGE - 运行的次数
